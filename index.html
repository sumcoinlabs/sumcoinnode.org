SumcoinNode - Deploy a Sumcoin supernode in seconds

What is SumcoinNode?

SumcoinNode is a small open source project. The project was inspired by a post on the SumcoinTalk forums by sumtoshi and aims to make it easy for individuals to deploy and maintain Sumcoin supernodes.


What is Sumcoin and what is a Sumcoin supernode?

Sumcoin is a peer-to-peer internet currency also known as crypto currency.

A Sumcoin supernode is a computer that is responsible for hosting and distributing the Sumcoin block chain and relaying Sumcoin transactions across the network.

Running a Sumcoin supernode supports the Sumcoin network and it's clients.


What do you do?

We make pre-made scripts individuals can run on their computers. The scripts will install and configure the Sumcoin software for the individual without prior knowledge of the software or how to install or configure it.

The scripts are currently in development and can be found on our GitHub page.

The general idea is that you can deploy a Sumcoin supernode with a simple one line command, the deployment script does the rest for you.
